Sample of Masters Supervision Topics
  • Money Laundering and AI - Exploring machine learning models to detect patterns of money laundering within financial transaction data.
  • Financial Crime and Big Data - Utilising big data techniques to analyse and detect financial crimes across large datasets.
  • Understanding DAG-based Cryptocurrencies - Analysing Directed Acyclic Graphs (DAGs) as an alternative to blockchain structures in cryptocurrencies.
  • Decentralised Identity Management Systems and Self-Sovereign Identity - Developing frameworks for identity verification systems that prioritise user sovereignty and decentralisation.
  • Building a Layer-1 Cryptocurrency - Designing and implementing a foundational cryptocurrency protocol, focusing on consensus mechanisms and security.
  • Building a Music App in Ethereum - Creating a decentralised application for music sharing and monetisation using Ethereum smart contracts.
  • Comparative Analysis of Bitcoin and Ethereum - Conducting a technical and functional comparison of the Bitcoin and Ethereum networks and their respective ecosystems.
  • Creating a Bitcoin Transaction Front End - Developing a user interface to facilitate the creation, management, and analysis of Bitcoin transactions.
  • Ransomware Analysis on the Bitcoin Transaction Set - Investigating Bitcoin transactions associated with ransomware attacks to identify key trends and patterns.
  • Bitcoin Transaction Capture and Analysis - Building tools to capture and analyse Bitcoin transactions for insights into transaction behaviour and network health.
  • Secure Timestamping Systems - Designing a timestamping system to verify the integrity and time of creation of digital documents using blockchain technology.
  • Implementing CryptDB - Implementing a database encryption solution that supports SQL queries over encrypted data, based on the CryptDB framework.
